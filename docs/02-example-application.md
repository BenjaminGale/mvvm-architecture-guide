# 2. Example Application

This repository contains a sample order management application that demonstrates the patterns described in the guide.

The application covers two feature areas — orders and customers — and shows how these are structured using the full pattern stack: domain models, services, repositories, use cases, ViewModels, sub-ViewModels, context objects, sessions, modules, and the composition root.

**Requirements:** Java 21 or later.

```bash
cd example && ./gradlew run
```
